ECS Library API

Publisher Detail

/publishers/<id>

URI Parameters

id integer The Identifier for the publisher (this is provided implicitly via the publisher's url when accessing the publisher collection).

Resource Properties

name string The registered company name of the publishing organisation
location string The location of the publisher's headquaters.
owner ECSID Read Only The user who created and owns the resource. Only this user may delete or modify the publisher.
url URI Read Only The location of this publisher resource on the server, including its ID.

OPTIONS

Returns a human readable name and description of endpoint and the availiable site-wide response and request content types: JSON, XML and CSV.

The description for this endpoint is blank.
