ECS Library API

Book List

/books

This resource is a collection of Book objects. New objects may be POSTed, however the list cannot be directly modified or replaced. If changes, deletions or replacements are required, these must be done on a per-publisher basis.

OPTIONS

Returns a human readable name and description of endpoint and the availiable site-wide response and request content types: JSON, XML and CSV.

The description for this endpoint is blank.

OPTIONS /books HTTP/1.1 Host: waisvm-rfp1.ecs.soton.ac.uk Accept: application/xml HTTP/1.1 200 OK Allow: GET, POST, HEAD, OPTIONS Content-Type: application/xml; charset=utf-8 <?xml version="1.0" encoding="utf-8"?> <root> <name>Book List</name> <description></description> <renders> <list-item>application/xml</list-item> <list-item>application/json</list-item> <list-item>text/csv</list-item> </renders> <parses> <list-item>application/xml</list-item> <list-item>application/json</list-item> <list-item>text/csv</list-item> </parses> </root>
