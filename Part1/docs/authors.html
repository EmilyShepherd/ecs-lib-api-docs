ECS Library API

Author List

/authors

This resource is a collection of Author objects. New objects may be POSTed, however the list cannot be directly modified or replaced. If changes, deletions or replacements are required, these must be done on a per-publisher basis.

OPTIONS

Returns a human readable name and description of endpoint and the availiable site-wide response and request content types: JSON, XML and CSV.

The description for this endpoint is blank.

OPTIONS /authors HTTP/1.1 Host: waisvm-rfp1.ecs.soton.ac.uk Accept: application/xml HTTP/1.1 200 OK Allow: GET, POST, HEAD, OPTIONS Content-Type: application/xml; charset=utf-8 <?xml version="1.0" encoding="utf-8"?> <root> <name>Author List</name> <description></description> <renders> <list-item>application/xml</list-item> <list-item>application/json</list-item> <list-item>text/csv</list-item> </renders> <parses> <list-item>application/xml</list-item> <list-item>application/json</list-item> <list-item>text/csv</list-item> </parses> </root>
OPTIONS /authors HTTP/1.1 Host: waisvm-rfp1.ecs.soton.ac.uk Accept: application/json HTTP/1.1 200 OK Allow: GET, POST, HEAD, OPTIONS Content-Type: application/json { "name" : "Author List", "description" : "", "renders" : [ "application/xml", "application/json", "text/csv" ], "parses" : [ "application/xml", "application/json", "text/csv" ] }
