ECS Library API

Author Detail

/authors/<id>

URI Parameters

id integer The Identifier for the author (this is provided implicitly via the author's url when accessing the author collection).

Resource Properties

name string The human name of this author.
owner ECSID Read Only The user who created and owns the resource. Only this user may delete or modify the author.
url URI Read Only The location of this author resource on the server, including its ID.

OPTIONS

Returns a human readable name and description of endpoint and the availiable site-wide response and request content types: JSON, XML and CSV.

The description for this endpoint is blank.
