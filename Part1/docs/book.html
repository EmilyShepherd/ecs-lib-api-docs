ECS Library API

Book Detail

/books/<id>

URI Parameters

id integer The Identifier for the book (this is provided implicitly via the book's url when accessing the book collection).

Resource Properties

title string The name of the book
pages integer The number of pages in the book
isbn13 ISBN13 The registered ISBN 13 for the book
date_published rfc3339 DATE The date the book was first published
authors Array<URI> Array of URLs identifying Author objects.
publisher URI Location of this book's Publisher object.
owner ECSID Read Only The user who created and owns the resource. Only this user may delete or modify the book.
url URI Read Only The location of this book resource on the server, including its ID.

OPTIONS

Returns a human readable name and description of endpoint and the availiable site-wide response and request content types: JSON, XML and CSV.

The description for this endpoint is blank.
